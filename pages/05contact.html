Why I chose this project

I thought it would be a great way to learn by teaching. Even though the road ahead is long, within time this website will be up and running, with actual topics to learn.

My first idea was a coffee shop

I misunderstood the task, and ended up creating a responsive website for a coffeeshop. But realised it had to be a website of a hobby, or somthing i enjoy. So i started from scratch, and thought a website about coding would be a great way for me to learn and improve my skills.

Too many ideas

One of the problems i faced with this task is that i had too many ideas. Should have sat down and done some brainstorming and few sketches, but i am far behind with my tasks, so decided to create ideas as i go. It was a bad idea!

Plan plan and plan

I have learnt so much working on this task. But if i had to pick out one thing i have leant to pass as advice, that would be: "Give your project time when you start, write psudo code, draw sketches and plan ahead, you'll thank yourself later!"
